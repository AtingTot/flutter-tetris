# flutter-tetris

基于Flutter的俄罗斯方块小游戏，支持Android与iOS。

# 玩法
底部三个按钮分别控制方块的左右移动和变形

**在游戏区域从上至下滑动屏幕可让方块迅速落下**

*排行榜功能未实现*
